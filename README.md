# TopBattleFR — Game Studio TikTok

App web autonome qui génère des vidéos TikTok (1080×1920, ~60 s) pour 5 jeux « zéro candidat » :
La Course, Trouve-le, Le Niveau 10, La Capture, Pause.

**Utilisation :** ouvrir `index.html` — export MP4 intégré (WebCodecs), partage natif vers la galerie / TikTok, mode plein écran pour enregistrement d'écran.

Le code source complet (fichiers séparés, README de production) vit dans le dépôt privé `gla` (`tiktok-game-studio/`).
Ce dépôt ne contient que le build publié.
